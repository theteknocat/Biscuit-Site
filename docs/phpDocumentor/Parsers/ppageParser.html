Parser::handleLogicBlock() - handler for LOGICBLOCK Logic Blocks are the stuff between { and } in a function/method. logic block can clearly contain other logic blocks, as in:
  1.  function test($a)
  2.  {
  3.     if (testcondition)
  4.     { // nested logic block
  5.     }
  6.  }
So, the exit portion of the logic block handler must check to see if the logic block being exited is the top-level, and it does this by retrieving the last event from the stack. then it backs up the word parser so that the function will exit properly.
