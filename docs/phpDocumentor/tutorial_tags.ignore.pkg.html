@ignore

Description

Use @ignore to prevent phpDocumentor from documenting an element, such as a duplicate element.

Example

Here is an example of how to use @ignore:

  1. if ($ostest)
  2. {
  3.     /**
  4.      * This define will either be 'Unix' or 'Windows'
  5.      */
  6.     define("OS","Unix");
  7. } else
  8. {
  9.     /**
  10.      * @ignore
  11.      */
  12.     define("OS","Windows");
  13. }
