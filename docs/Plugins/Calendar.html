Standalone calendar widget allowing for day and week selection and custom Javascript callback function. To use it as a date selector requires the existence of a page called "date_selector" in the page_index table. This ensures that the calendar only renders when called for specifically, so it cannot interfere with the rendering of other AJAX requests.

Render a date field with an actuator button

Parameters

  • string $fieldname: Name/ID of the form text field to contain the date
  • string $date_format: Optional - How the date in the form-field should be formatted. Defaults to d-M-Y (eg. 20-Jan-2008). See the PHP manual on the date() and strtotime() functions if you need help determining what formats can be used.
  • string $default_date: Optional - The date you want to appear in the form field by default. This can be in any string date format. It will be converted to the format provided by $date_format regardless
  • string $css_class: Optional - A CSS classname to add to the text field
  • string $custom_btn_img: Optional - an image to use for the actuator button. If left blank it will display a standard form button labelled "Pick"
