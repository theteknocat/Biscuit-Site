A wrapper for the HTML purifier library.


A wrapper for the HTML purifier library.  When this class gets instantiated, it determines which version of the purifier library to include based on the PHP version, instantiates it (storing it in it's own property for easy re-use), and can then call it with wrapper functions that allow a list of allowable HTML tags on any individual purifier call
