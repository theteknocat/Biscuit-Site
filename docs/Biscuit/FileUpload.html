Handle a file upload.


Handle a file upload.
    $uploaded_file = new FileUpload($_FILES["publication_file"], "/uploads/publications")    if ($uploaded_file->is_okay()) {        $filename = $uploaded_file->file_name;        ...    } else {        $error = $uploaded_file->get_error_message();        ...    }

		
Parameters

	
    
			
  • array $uploaded_file: Required. Contains the file data (must resemble $_FILES['foo'])
    • 
			
  • string $upload_dir: Required. File upload directory relative to the web root (ie. "/images/photos").  If the directory doesn't exist, it will be created
    • 
			
  • string $type: Optional. If left blank (default), the file will simply be uploaded to the $upload_dir.
    • 
			
  • string $img_width: If type=image, enter this to override the system-wide global "IMG_WIDTH" for resizing the uploaded image to it's medium/large size
    • 
			
  • string $img_height: If type=image, enter this to override the system-wide global "IMG_HEIGHT" for resizing the uploaded image to it's medium/large size
    • 
			
  • string $thumb_width: If type=image, enter this to override the system-wide global "THUMB_WIDTH" for resizing the uploaded image to it's thumbnail size
    • 
			
  • string $thumb_height: If type=image, enter this to override the system-wide global "THUMB_HEIGHT" for resizing the uploaded image to it's thumbnail size
