Return the request type. to render are:

Return the request type. Request types in Biscuit are primarily for determining whether or not it should render output. The two types that it checks when asked to render are:
 standard - normal page request, noting special update - an Ajax update request
 You can submit any other value you want in the request_type parameter with any page request and use it however you want with plugins that perform actions that do not need to render any output.
